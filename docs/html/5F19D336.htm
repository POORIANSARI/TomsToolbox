DefaultItemCommand The default item command is the command that will be executed when an item of the items control has received a mouse double click or enter key. It is not executed when the double-click is on the background or on the scrollbar. This command avoids the ubiquitous wrong implementations as well as code duplication when handling double-clicks in items controls like the ListBox

The command parameter for the command is the item that has been clicked.

RefreshOnSourceChanges The object that will be observed for changes. A change of the object will trigger a refresh on the collection view of the attached items control.
