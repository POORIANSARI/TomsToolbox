TextBlockHelper Class

Attached property provider which adds the read-only attached property IsTextTrimmedProperty to the framework's TextBlock control. Can be used to e.g. show a dynamic tool tip with the full text, that shows up only if the text is really trimmed.

IsAutomaticToolTipEnabled Set this property to true to enable automatic evaluation of the IsTextTrimmed attached property. This property is used by the style identified with the AutoToolTipTextBoxStyle to display of a tool tip only if the text of the text block is trimmed.
IsTextTrimmed If the IsAutomaticToolTipEnabled attached property is set to true on a text block, this property reflects if the text inside the text block is trimmed, i.e. not fully visible.
