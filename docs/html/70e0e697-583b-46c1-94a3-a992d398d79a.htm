Welcome to Tom's Toolbox

A rich set of functions and classes to ease every days .Net development tasks.

The modules are targeting Net6.0, Net7.0, .NetFramework 462, and NetStandard 2.1, depending on the functionality they provide

All code is annotated with Nullable reference types , so you can easily build robust code.

Symbol files with direct references to the GitHub sources are included, so you can easily debug the libraries.

Getting Started

  • Browse this documentation to find out how this library can support you.

  • Use NuGet to integrate the library into your projects.

  • Browse or download the source code to learn about the internals or integrate the parts of interest with your own tool set.
